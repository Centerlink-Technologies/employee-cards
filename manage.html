Add Employee

  1. Go to index.html and fill in the employee form.
  2. Download the generated ZIP file (e.g., mike-smith-employee-card.zip).
  3. Unzip the folder locally.
  4. Upload the employee folder (e.g., mike-smith/) to GitHub Pages at /employee-cards/.
  5. Add the slug (e.g., "mike-smith") to the EMPLOYEE_SLUGS array in main.js.
  6. Commit and push the changes.

Remove Employee

To remove an employee from the directory:

  1. Delete the employee folder from GitHub Pages. Example: /employee-cards/mike-smith/
  2. Remove the slug from the EMPLOYEE_SLUGS array in main.js. Example: Remove "mike-smith"
  3. Commit and push the changes.
  4. The employee will no longer appear on the directory.
